# Webpack + Babel
`babel-loader` embeds `Babel` compiler into bundling process

## To run:
- install packages
```
npm i
```
- run
```
npm run build
```
- ensure that the code is transpiled to support the oldes browser (IE 9)
- remove IE support, delete in `.babelrc`:
```
"last 2 IE versions"
```
- check how the code was updated
